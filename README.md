### `Run Locally`

+ Clone the project:

      git clone https://github.com/satishshinde2048/MinishopApp-Reduxasync/.git


+ Navigate to the project directory:

      cd demoapp
                                                                                                       
+ Install dependencies:                                                                                                    

       npm install 
                                                                                                                                                                                                                
+ Start the server:                                                                                                        

       npm run start
